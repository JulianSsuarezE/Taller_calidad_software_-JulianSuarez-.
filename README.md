# Proyecto Final - Nombre del Proyecto

## Descripción
Este proyecto consiste en una aplicación web de gestión de productos de una piñateria,
desarrollado en PHP con base de datos MySQL.

## Requisitos para ejecutarlo
- XAMPP (con Apache y MySQL)
- PHP
- Navegador web (Google Chrome, Fire Fox, etc)

---

## Instalación
1. Copiar el proyecto dentro de la carpeta `xampp/htdocs` y pegar la carpeta `piñateria`.
2. Crear una base de datos llamada: `pinateria_db`
3. Importar el archivo SQL que está en: `/database/pinateria_db.sql`
4. Verificar la conexión en `config.php`, que el nombre sea igual a la de la BD (`pinateria_db.sql`)
5. Abrir en el navegador de su preferencia.

Si tiene problemas con la base de datos, revise el archivo:
piñateria/database/instrucciones.txt
